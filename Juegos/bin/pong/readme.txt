Atajos de teclado:

El azul se mueve con las flechas para arriba y para abajo, el rojo con las teclas w y s.

Para iniciar el movimiento de la pelota se debe apretar el space.

Para reiniciar el juego se toca la r.

Para bajar un punto al jugador rojo (player 1), se pulsa el numero 1.
Para bajar un punto al jugador azul (player 2), se pulsa el numero 2.

Exit para salir
